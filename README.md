# USA Reverse Phone Lookup

![image](https://raw.githubusercontent.com/thepythoncode97/USA-Reverse-Phone-Lookup/refs/heads/main/phone-reverse-lookup.png)

This software allows you to perform a reverse phone lookup for the **USA Only**, providing the following details:

1. **Get Name**: Retrieves the user's full name for personalization or identification purposes.
2. **Get Age**: Captures the user's age, which may be used for eligibility checks or demographic analysis.
3. **Get Address**: Collects the user's complete street address for contact, delivery, or verification needs.
4. **Get City**: Retrieves the city in which the user resides.
5. **Get State**: Collects the user's state or province information.
6. **Get Zip**: CodeGathers the user's ZIP or postal code, used for geographic identification or shipping calculations.
7. **Get Birth**: YearRetrieves the user's year of birth. This can be used to calculate age or verify age-related requirements.
